## 支持功能
1. 小爱同学操控(无外部按键,blinker接入)
2. 流光溢彩/音效律动效果(电脑端下载Prismatik软件,选择adalight)
3. 显示屏(一言/西安疫情实时人数/天气/时钟)
4. 久坐提醒(阴间红色)
## 注意事项 
1. 灯带不必外接电容/电阻
2. 务必保证所有电源共地(esp32/灯带供电)
3. 禁止用esp32对灯带供电,灯带电流约5A,会烧的
4. 务必保证灯带不进水,走线不碰水
5. 保证你的esp32有多余引脚进行配置(3个或以上)
